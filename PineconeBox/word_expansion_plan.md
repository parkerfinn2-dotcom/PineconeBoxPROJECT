## 4. 技术实现

### 4.1 词汇导入工具
- **脚本**: `backend/import_words.py`
- **功能**:
  - 从CSV文件导入词汇
  - 自动分配难度等级
  - 支持词汇去重
  - 生成词汇统计报告

### 4.2 词汇文件格式
```csv
word,meaning,category,source
apple,苹果,日常用品,raz_aa_c
banana,香蕉,日常用品,raz_aa_c
```

### 4.3 批量导入步骤
1. **准备词汇文件**:
   - 按主题和等级整理词汇
   - 保存为CSV格式

2. **运行导入工具**:
   ```bash
   cd backend
   python3 import_words.py
   # 选择选项3，输入CSV文件路径
   ```

3. **验证导入结果**:
   ```bash
   python3 import_words.py
   # 选择选项5，查看统计结果
   ```
